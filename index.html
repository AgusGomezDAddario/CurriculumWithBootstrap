Conocimientos

Estas son algunas de las herramientas que utilizo en mis proyectos

Drupal

Es una plataforma de código abierto que se emplea para constuir sitios web dinámicos. Además, permite la gestión de contenidos y personalización mediante módulos.

Herramienta usada en el Proyecto del Sitio Web para la carrera Ingeniería en Sistemas

React

Es una biblioteca de JavaScript que se emplea para crear interfaces de usuario en la web. Además, facilita la creación de componentes reutilizables y actualización dinámica.

Herramienta usada en el Proyecto del Sitio Web del médico en Memorium

React Native

Es una biblioteca usada para desarrollar aplicaciones móviles en Android e iOS. Permite compartir código entre ambas plataformas y utiliza componentes nativos en todo tipo de dispositivos.

Herramienta usada en el Proyecto de la Aplicación en Memorium

Arduino

Es una plataforma de prototipado electrónivo de código abierto que permite crear objetos electrónicos interactivos con lenguaje C/C+. Actúa controlando las placas Arduino, cuyo uso es muy amplio en la industria.

Herramienta usada en el Proyecto Mano Robótica en el laboratorio Codapli
